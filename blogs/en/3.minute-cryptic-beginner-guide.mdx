## What Is Minute Cryptic?

Minute cryptic is a daily one-clue format. You solve one short cryptic clue with optional hints and then review the explanation.

## The Core Idea

Each clue has:

- a definition
- a wordplay construction
- an answer length

The correct answer must satisfy all three.

## Your First Solving Workflow

1. Mark likely definition at clue start or end.
2. Identify indicator words.
3. Build candidate answers from wordplay.
4. Filter by letter count.
5. Confirm definition and structure both fit.

## How to Use Hints Correctly

Hints should unblock, not replace solving.

- reveal one hint level
- attempt full solve again
- only reveal next level if still blocked

## Common Beginner Errors

- trusting a candidate that fails definition
- ignoring letter count
- sticking to one parse too long
- revealing deep hints too early

## One Week Improvement Plan

- Solve one clue daily.
- Record one indicator word per clue.
- Read explanation even after correct answers.
- Revisit one older clue from the archive each week.

## Final Takeaway

You do not need advanced vocabulary to start. You need a reliable process and steady daily reps.
