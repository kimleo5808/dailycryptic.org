# Content Plan

## Purpose

This document translates the approved roadmap in [AGENTS.md](/F:/网站/游戏网站/dailycryptic.org/AGENTS.md) into a page-by-page content production brief.

Use this file when:

- writing page copy
- selecting puzzle examples
- reviewing whether a page is distinct enough to publish
- checking whether a page meets the minimum SEO/content standard

## Global Content Rules

- Every new page must be at least 1000 words
- Preferred content range is 1200 to 2200 words
- Every page must answer the search query in the first 2 paragraphs
- Every page must include H2 and H3 structure
- Every page must include at least one FAQ block
- Every page must include at least 3 internal links
- Every page must include at least one playable puzzle route
- Every page should use real puzzle examples from the site where possible
- Every page should include a "what to do next" or "practice next" section

## Voice And Positioning

The tone should be:

- practical
- clear
- instructional
- non-academic
- non-salesy

The pages should feel like useful puzzle-learning resources, not keyword landing pages.

Avoid:

- filler paragraphs
- repetitive intros
- generic SEO wording
- exaggerated claims
- AI-sounding transition spam

## Example Selection Rules

When choosing site puzzles as examples:

- prefer clean and representative examples over flashy ones
- avoid using the same example on too many pages
- use easier examples on beginner-facing pages
- use exact clue-type matches on clue-type pages
- ensure every example explanation is correct and easy to restate

## FAQ Rules

Each page should:

- include 4 to 6 FAQs
- use search-like question wording
- avoid vague marketing language
- answer directly within 2 to 4 sentences

## Content Production Checklist

Use this checklist for each page:

- [ ] Primary keyword confirmed
- [ ] Secondary keywords confirmed
- [ ] Target word count set
- [ ] H2 outline approved
- [ ] H3 outline approved
- [ ] FAQ questions approved
- [ ] Internal link targets selected
- [ ] Puzzle examples selected
- [ ] Intro answers query directly
- [ ] Ending includes next-step practice route
